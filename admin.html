Quick Guide
  1. Click on a section in the sidebar to edit its content
  2. Fill in or change the text fields, image URLs, etc.
  3. Click Save All to save changes (stored in your browser)
  4. Click Preview in the sidebar to see a live preview
  5. When done, click Export HTML to download the final website code
  6. Upload the exported HTML to Wix (see Wix Guide below)
Wix Deployment Guide Important

Option A: Wix HTML Embed (Recommended for custom code)

  1. Export HTML from this admin panel
  2. Log in to your Wix editor
  3. Add an "Embed HTML" element (Add → Embed → Custom Embeds → Embed a Widget)
  4. Paste the exported HTML code
  5. Resize the widget to fill the full page width
  6. Publish your Wix site

Option B: Wix Velo (For full custom site)

  1. Enable Velo by Wix in your editor (Dev Mode)
  2. Use the exported HTML structure as a reference
  3. Recreate sections using Wix elements + custom CSS/JS in Velo

Option C: External Hosting + Wix Domain

  1. Upload the HTML file to Netlify, Vercel, or GitHub Pages (free)
  2. Connect your forestdayspa.com domain to the hosting provider
  3. This gives full control and the best performance
